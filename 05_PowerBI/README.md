# 05_PowerBI – Analytical Dashboard

This folder contains the Power BI dashboard developed as the main analytical artifact of the dissertation project.

The dashboard supports the analysis of teachers’ and school heads’ salary evolution in Portugal in relation to key macroeconomic indicators between 2021 and 2024.

---

## Dashboard objectives

The dashboard was designed to support:
- Longitudinal salary analysis
- Inflation-adjusted salary comparison
- Macroeconomic contextualization
- Cross-dimensional exploration
- European comparative analysis

---

## Main analytical areas

### Salary evolution
Analysis of nominal and real salary evolution across:
- Teachers
- School heads
- Career stages
- Qualification levels
- ISCED levels

### Macroeconomic dynamics
Integration of indicators such as:
- GDP per capita
- Consumer Price Index
- Average annual wages
- Housing-related indicators
- Education expenditure

### European comparison
Cross-country comparison using:
- Statutory salaries
- Purchasing Power Standard (PPS)
- Relative European positioning

---

## Dashboard features

The dashboard includes:
- Interactive filtering
- KPI indicators
- Dynamic visualizations
- Comparative trend analysis
- Cross-dimensional exploration
